# Murmur

## Test Blindness Policy

- NEVER read, clone, browse, or access the `murmurlinux/tests` repository
- NEVER use GitHub MCP tools to read files from `murmurlinux/tests`
- NEVER ask about or reference the contents of blind tests
- If blind tests fail in CI, diagnose from error messages only, not test source code
- This policy exists to prevent AI-optimised code that passes tests without being genuinely correct
